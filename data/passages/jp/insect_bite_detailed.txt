벌레 물림 상세 응급처치 가이드

【벌레 물림의 종류】
1. 모기 물림
   - 증상: 가려움증, 작은 부종, 발적
   - 치료: 항히스타민 연고, 냉찜질
   - 주의: 긁지 않기, 감염 방지

2. 벼룩 물림
   - 증상: 작은 빨간 점, 가려움증
   - 치료: 소독, 항히스타민 연고
   - 주의: 옷과 침구 세탁

3. 진드기 물림
   - 증상: 작은 붉은 반점, 가려움증
   - 치료: 핀셋으로 제거, 소독
   - 주의: 라임병 감염 가능성

4. 거미 물림
   - 증상: 통증, 부종, 발적
   - 치료: 냉찜질, 소독
   - 주의: 독거미 여부 확인

【응급처치 단계】
1. 즉시 조치
   - 해당 부위를 깨끗한 물로 씻기
   - 비누 사용하여 세척
   - 건조한 수건으로 가볍게 닦기

2. 부종 완화
   - 얼음팩이나 차가운 물수건 사용
   - 15-20분간 냉찜질
   - 하루에 3-4회 반복

3. 가려움증 완화
   - 항히스타민 연고 바르기
   - 칼라민 로션 사용
   - 긁지 않도록 주의

4. 감염 방지
   - 상처 부위를 깨끗하게 유지
   - 소독제 사용
   - 붕대나 반창고로 보호

【의료진 상담이 필요한 경우】
- 24시간 후에도 개선되지 않는 경우
- 부종이 심해지는 경우
- 발열이나 오한이 있는 경우
- 상처 부위가 빨갛게 변하거나 고름이 나는 경우
- 전신에 두드러기가 생기는 경우
- 호흡곤란이나 현기증이 있는 경우

【예방법】
- 벌레 퇴치제 사용
- 긴 옷 착용
- 밝은 색 옷 피하기
- 향수나 화장품 사용 자제
- 벌레가 많은 곳 피하기
- 침구와 옷을 자주 세탁

【주의사항】
- 벌레 물림은 대부분 가벼운 증상이지만, 감염이나 알레르기 반응이 있을 수 있음
- 특히 어린이나 노인, 면역력이 약한 사람은 더 주의 필요
- 이전에 심한 알레르기 반응이 있었던 경우 즉시 의료진 상담
