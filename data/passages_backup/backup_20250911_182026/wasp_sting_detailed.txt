말벌 쏘임 상세 응급처치 가이드

【말벌 쏘임의 위험성】
1. 즉시 반응
   - 통증, 부종, 발적
   - 가려움증, 따끔거림
   - 침이 남아있을 수 있음

2. 알레르기 반응
   - 두드러기, 발진
   - 얼굴이나 목 부종
   - 호흡곤란, 목소리 변화
   - 현기증, 의식 잃음
   - 아나필락시스 쇼크

3. 생명 위험
   - 기도 부종으로 인한 질식
   - 혈압 저하로 인한 쇼크
   - 심장 마비 가능성

【응급처치 단계】
1. 즉시 조치 (1-2분 내)
   - 침 제거 (핀셋이나 신용카드 가장자리 사용)
   - 상처 부위를 깨끗한 물로 씻기
   - 비누 사용하여 세척

2. 부종 완화 (5-10분 내)
   - 얼음팩이나 차가운 물수건 사용
   - 15-20분간 냉찜질
   - 상처 부위를 심장보다 높게 유지

3. 통증 완화
   - 항히스타민 연고 바르기
   - 진통제 복용 (의사 처방 필요)
   - 부드러운 마사지 (심하지 않게)

4. 모니터링
   - 30분간 증상 변화 관찰
   - 알레르기 반응 징후 확인
   - 호흡 상태 체크

【즉시 119 신고가 필요한 경우】
- 호흡곤란이나 목소리 변화
- 얼굴이나 목 부종
- 전신 두드러기나 발진
- 현기증이나 의식 잃음
- 이전에 말벌에 쏘인 적이 있는 경우
- 에피네프린 자동주사기(에피펜) 사용 후

【알레르기 반응 대응】
1. 에피네프린 자동주사기 사용
   - 대퇴부 외측에 주사
   - 10초간 누르고 있기
   - 즉시 119에 연락

2. 항히스타민제 복용
   - 의사 처방이 필요한 경우
   - 응급실에서 처방받기

3. 응급실 즉시 방문
   - 알레르기 반응이 있는 경우
   - 증상이 악화되는 경우

【예방법】
1. 환경 관리
   - 벌집 근처 피하기
   - 밝은 색 옷 피하기
   - 향수나 화장품 사용 자제

2. 행동 주의
   - 갑작스러운 움직임 피하기
   - 벌을 쫓거나 공격하지 않기
   - 달콤한 음료나 음식 주의

3. 보호 장비
   - 긴 옷 착용
   - 모자 착용
   - 벌레 퇴치제 사용

【주의사항】
- 말벌 쏘임은 생명을 위협할 수 있는 심각한 상황
- 특히 이전에 알레르기 반응이 있었던 경우 더욱 주의
- 어린이나 노인, 만성질환자는 더 위험
- 응급처치 후에도 반드시 의료진 상담 필요
- 에피네프린 자동주사기를 소지하고 있는 경우 즉시 사용
